* Go-Snake
Snake written in go with [[https://github.com/raysan5/raylib][raylib]] [[https://github.com/tuxikus/go-snake][go bindings]].

* How to run
#+begin_src shell
  git clone https://github.com/tuxikus/go-snake
  cd go-snake
  direnv allow # if using flake
  go run .
#+end_src

* Controls
| *Key*    | *Action*   |
| ESC    | Exit     |
| Enter  | Select   |
| Arrows | Movement |
